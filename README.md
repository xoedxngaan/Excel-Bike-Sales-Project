# Excel-Bike-Sales-Project
This project analyses a bike sales dataset sourced from Kaggle using Microsoft Excel as a traditional data analysis tool. The aim of the project was to explore purchasing patterns and determine which demographic groups were more likely to purchase a bike across different countries and regions.

The dataset was first cleaned and prepared for analysis by identifying and removing duplicate records and organising the data into a structured format suitable for analysis.

Once the data was prepared, Excel was used to perform exploratory data analysis to understand how factors such as age group, income level, education, marital status, and region influence bike purchasing behaviour.

# Analysis Focus
The project specifically examines:
• Which age groups are most likely to purchase bikes

• How income levels and education backgrounds influence purchases

•Differences in bike purchases across countries and regions

•The average sales and profit trends across different geographic areas

# Tools and Techniques Used:
# Microsoft Excel for the following:

• Data cleaning and preparation

• Duplicate detection and removal

• Pivot table analysis for demographic insights

•Interactive dashboard creation for visual analysis

# Project Output
The final Excel workbook includes:
• A cleaned working dataset

• Multiple pivot tables used to summarise and analyse demographic and regional trends

• An interactive dashboard allowing users to filter bike purchases by country, region, education level, marital status, age group, and other demographic variables

• The dashboard enables quick visual exploration of how different demographic characteristics influence bike purchasing behaviour.
